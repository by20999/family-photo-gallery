# `server/middleware/` 中间件说明

本目录保存 Express 全局中间件，负责横切能力，不承载具体照片、分组或故事业务逻辑。

## 模块清单

| 文件 | 作用 |
| --- | --- |
| `auth.js` | 保护 `/api` 写入请求，要求提供管理密码。 |
| `writeQueue.js` | 串行化 `/api` 请求，降低文件型 JSON 存储的并发覆盖风险。 |

## 设计约束

- 中间件应尽量无业务状态，避免直接读写照片、分组、故事数据。
- 鉴权失败统一返回 403 和 JSON 错误信息。
- 当前管理密码是轻量保护，不等同于完整账号体系。
- 文件存储模式下，部分 GET 请求可能补缩略图或同步元数据，因此 `/api` 队列覆盖所有 API 请求。
