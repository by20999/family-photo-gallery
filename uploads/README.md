# `uploads/` 上传文件目录

本目录保存用户上传的原图。默认缩略图保存在 `uploads/thumbnails/`。

## 重要说明

- 这里是运行数据，不是源码。
- 不要在未备份的情况下删除本目录内容。
- 照片文件名就是系统中的照片 ID。
- `photo-data.json` 通过文件名关联这里的图片。

## 子目录

| 路径 | 作用 |
| --- | --- |
| `thumbnails/` | 本地开发或未设置 Volume 时的缩略图目录。 |

## 部署注意

如果设置了 `RAILWAY_VOLUME_MOUNT_PATH`，上传目录会改为该环境变量指向的位置，而不是项目内 `uploads/`。

## 备份要求

备份相册时必须包含：

- 本目录下的原图。
- 缩略图目录。
- 根目录或 Volume 中的 JSON 数据文件。
