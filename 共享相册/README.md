# 家庭共享相册

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
npm start
```

### 3. 访问网站
打开浏览器访问: http://localhost:3000

## 如何分享给家人朋友

### 方案一：局域网内分享（同一 WiFi）
1. 启动服务器后，查看你的本机 IP 地址
   - Windows: 打开命令行输入 `ipconfig`，找到 IPv4 地址
   - Mac/Linux: 输入 `ifconfig` 或 `ip addr`
2. 让家人访问: http://你的IP地址:3000
   - 例如: http://192.168.1.100:3000

### 方案二：部署到云服务器（推荐）
可以部署到以下平台，获得永久网址：
- **Vercel** (免费，简单)
- **Railway** (免费额度)
- **阿里云/腾讯云** (需要购买服务器)

## 功能说明
- ✅ 上传图片（支持多选）
- ✅ 查看所有人上传的图片
- ✅ 点击图片全屏预览
- ✅ 删除图片
- ✅ 图片永久保存在服务器

## 注意事项
- 图片保存在 `uploads` 文件夹
- 单张图片最大 10MB
- 一次最多上传 10 张
