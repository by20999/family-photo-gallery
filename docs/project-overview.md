# 项目总览

## 一句话说明

家庭共享相册是一个基于文件系统的轻量照片站点，用于保存、整理和回看家庭照片，并通过评论、表情、分组和故事流增强回忆叙事。

## 用户场景

- 家庭成员上传旅行、节日、日常照片。
- 按标签、月份、分组或收藏快速查找照片。
- 给照片补充描述、评论和表情回应。
- 把一组照片整理成故事流，用时间线方式回看。
- 部署到个人服务器、局域网或支持持久化 Volume 的平台。

## 当前功能边界

项目已经具备：

- 单页前端应用。
- Express 静态资源和 API 服务。
- 图片上传、缩略图生成和原图访问。
- 照片元数据、分组数据、故事数据的 JSON 存储。
- `/api` 写入接口的基础管理密码保护。
- 单进程 API 队列，降低 JSON 并发写入互相覆盖的风险。
- 健康检查、冒烟测试、数据体检和本地备份脚本。

项目尚未具备：

- 用户账号体系。
- 权限分级。
- 数据库。
- 后台管理台。
- 云对象存储。
- 多实例并发写入保障。
- 完整端到端测试。

## 质量目标

- 小规模可靠：适合个人或家庭低并发使用。
- 易恢复：数据文件和上传目录清楚可备份。
- 易接手：目录说明、模块职责、API 和数据模型有明确文档。
- 易演进：后续可逐步迁移到数据库、对象存储或账号系统。

## 关键限制

- JSON 文件存储不适合高并发写入。
- 管理密码只是轻量保护，不等同于认证授权系统。
- 图片上传依赖本地磁盘或挂载 Volume，部署环境必须保证持久化。
- 用户已确认旧相册数据不保留，当前工作区已清空旧照片元数据，可作为新相册基线继续开发。
- 当前 CSS 和部分前端模块体积较大，后续维护需要谨慎拆分。
