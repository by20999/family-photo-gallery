# 项目改进清单

- 修复剩余历史乱码文本，统一 UTF-8。
- 增加数据层单元测试和浏览器端冒烟测试。
- 为上传文件增加更严格的真实图片解析校验和失败清理。
- 增加恢复脚本或恢复演练文档。

## 中优先级

- 继续拆分 `js/gallery.js`，本轮已先把系统状态面板独立到 `js/system.js`。
- 把批量整理从连续 prompt 升级为专用弹窗，减少误操作。
- 为 `story.js` 增加更清晰的内部结构。
- 为 API 增加统一错误码。
- 增加结构化日志和错误追踪。
- 增加缩略图重建/清理脚本。

## 低优先级

- 增加相册导出功能。
- 重复照片检测已完成基础版；后续可增加相似图片检测和重复照片批量处理界面。
- 增加按人物、地点、事件的智能归类。
- 增加更多主题包。

## 技术债

- 当前只有基础冒烟测试，缺少系统化测试。
- 当前无 lint/format。
- 当前没有账号体系，只有轻量管理密码。
- 当前 JSON 存储不适合高并发。
- 当前大文件模块影响长期维护效率。
